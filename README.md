# Book Borrow System
## Python控制台图书借阅管理工具
练习嵌套列表存储多维度信息、状态字段判断、图书借还业务逻辑，文本持久化存储。

### 功能
1. 添加图书：录入书名，默认状态在馆、无人借阅
2. 借书功能：校验图书是否已被借出，登记借阅人
3. 还书功能：重置借阅人为无、状态恢复在馆
4. 列表格式化展示所有图书，清晰区分借阅状态
5. 完整输入校验、异常捕获，防止程序崩溃
6. 数据永久保存在 book.txt，重启不丢失

### 核心知识点
1. 字典嵌套列表存储多组数据：{书名:[借阅人,状态]}
2. 字符串分隔符 `|` 多行读写文件
3. 状态字符串判断（在馆/已借出）实现业务逻辑
4. f-string固定宽度对齐打印表格
5. 文件w覆盖写入持久化更新数据
6. while菜单循环、多分支elif功能选择
7. try-except捕获非数字菜单输入

### 拓展方向
1. 增加借阅期限，超期提醒
2. 支持批量添加多本图书
3. 按借阅人筛选借出书籍
4. 增加删除图书功能
